Member - Select Specialist

This section is a prototype interface to choosing a specialist. The interface is particularly simple for the moment, but eventually want to be able to have texts being typed in and the system will look at the list of words and determine what kind of search is being requested. At first, we may simply use the set of phrases that will be matched and used to populate what we are looking for. But now we would use just a set of drop downs to pretend that, we've had this translation from text to a search request.

The purpose of this page is to give an idea of what the final interface may look like. The idea behind this page is that there's an input box where remember, can enter free text, and the system will determine what kind of specialist they use a may need to visit and give advice on the next steps. When no advice can be given, it would normally suggest that you see your primary care physician.

For now to drop down list should be sufficient, but in the future, we will use free text to select from the two drop-down lists.
