Stack!!

A stack is exactly what it sounds like. An element gets added to the top of the stack and only the element on the top may be removed. This is an example of an array implementation of a Stack. So an element can only be added/removed from the end of the array.

Functions: push, pop, peek, view, length

StackES6

Stack data-structure.

It's work is based on the LIFO method (last-IN-first-OUT).

It means that elements added to the stack are placed on the top and only the last element (from the top) can be reached. After we get access to the last element, he pops from the stack.

This is a class-based implementation of a Stack. It provides functions 'push' - to add an element, 'pop' - to remove an element from the top. Also it implements 'length', 'last' and 'isEmpty' properties and static isStack method to check is an object the instance of Stack class.
