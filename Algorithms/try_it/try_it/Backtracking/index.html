Backtracking

Introduction

This directory consists of a set of algorithms to backtracking algorithms. Backtracking

Some problems can be solve through a set of strial end error steps hen a choice has to taken from a ste of possible moves. Backgracking involves trying each of the possible actions until a solution is found. Typically this in implemented recursively, at each level try and an action to see if it give the desired desired solution. If not try the next, until all options are tried out, or the computation limit is reached.

The algorithm is listed in this directory are:

Knight's Tour

On a chessboard find a set of moves for a knight such that it visits every square on the board. Ideally every square should be visited only once.

Place N Queens

On a N x N chessboard place N queens on the boad so that no queen can take the other.
