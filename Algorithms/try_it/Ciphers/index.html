Ciphers

Introduction

Some extremely simple excryption algorithms that can hide a mesage (data) from all other than the intended recepient. These are not at all useful in real life, since they are extremely easy to break.

Julius Caesars Cipher

This is an encryption method believed to have been used by Julius Caesar. It is a classic character replacement strategy, where each characted is uniquely replaced by another one. It is very easy to break.

CaesarsCipher

Caesar's Cipher - also known as the ROT13 Cipher is when a letter is replaced by the one that is 13 spaces away from it in the alphabet. If the letter is in the first half of the alphabet we add 13, if it's in the latter half we subtract 13 from the character code value.
