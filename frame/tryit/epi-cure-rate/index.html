Episode of Care - Epi Cure Rate

Introduction

This is an introduction to an exciting new application for creating and understanding episodes of care. EPI_CURE_RATE. The primary benifits of the tool is the interactive realtime nature of the functions it provides.

This application to anylyze vast number of historical claim for patients (members) and catorgerize the conditions episode of care (episode) that into which the members can be placed. This is also known as claim bundling. The keys ideas behind this work are as follows

  • Rules to determine if a member is in an episode. This is refered to as a trigger rule.

  • Rules to determine if a service (CPT code, ICD procedure code, revenue code …) should be included in the episode

  • Rules to determine the other requirements for the rules (place of service, the type of claim, age, sex …)

  • Provides a place to edit and understand the rules

  • Apply the rules to vast amount of historical data very quickly

  • Tools to analyze the resulting bundes to determine the following: * Total number of episodes * The cost for each member. * The total cost for all members in the episode * The frequency of the vatious service procidec * …
